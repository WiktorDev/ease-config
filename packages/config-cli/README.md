# @ease-lab/config-cli

CLI tool for `@ease-lab/config`.

Generates TypeScript types from your `config/` directory.

---

## 📦 Installation

```bash
npm install -g @ease-lab/config
ease-config generate
```
or use via npx:
```bash
npx ease-config generate
```

## 🔄 Workflow
After creating or updating config files, run:
```bash
ease-config generate
```

## 🧠 Purpose
This CLI keeps your configuration types in sync with your `config/` files.
